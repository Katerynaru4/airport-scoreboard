# Airport scoreboard

**[Site link](https://katerynaru4.github.io/airport-scoreboard/departures)**

Airport scoreboard page supports mobile(340px), tablet(up-to-1024px) and desktop (up-to-1440px) versions

### The tech stack is:
- React
- Redux
- JavaScript
- Redux
- Sass (Scss)
- REST API
